Página web de local de hamburguesas

Este proyecto consiste en una página web estática, que presenta un menú de hamburguesas con un estilo visual atractivo y moderno. La página incluye secciones con información sobre los productos, precios y una interfaz limpia que simula la experiencia de una landing page para un local gastronómico. Tecnologias utilizadas: HTML y CSS (Flexbox, Grid, animaciones básicas)
